# Screenshots Directory

This folder contains screenshots of the Flutter DApp and Ganache blockchain explorer.

## How to Add Images

1. **Flutter App Screen** (`flutter_app_screen.png`)
   - Screenshot of the Flutter app running in Edge browser
   - Shows the "Hello zedamine" greeting and input form
   - Save the Flutter app UI screenshot here

2. **Ganache Transaction** (`ganache_transaction.png`)
   - Screenshot of Ganache showing transaction details
   - Shows the setName transaction with gas usage and block number
   - Save the Ganache explorer screenshot here

## Image Files Needed

- `flutter_app_screen.png` - Flutter UI screenshot
- `ganache_transaction.png` - Ganache explorer screenshot

These images will be automatically referenced in the main README.md file.
